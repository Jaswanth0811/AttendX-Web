Monorepo Root Directory Configuration Detected

If you are seeing this page, it means your hosting provider (e.g., Render, Vercel, etc.) is deploying or serving the root folder of your repository instead of the frontend subfolder.

To fix this and load your Next.js application, please configure the Root Directory to frontend in your hosting dashboard settings:

  1. Go to your hosting provider's dashboard (e.g., Render or Vercel).
  2. Select your frontend web service or project.
  3. Navigate to its Settings.
  4. Locate the Root Directory config and set it to frontend.
  5. For Render Static Sites, also set the Build Command to npm run build and Publish Directory to out.
  6. Save the changes and trigger a new deployment (redeploy or push a commit).

Once configured correctly, your Next.js application will compile and serve from the root URL!
